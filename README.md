📄 Consulta de CNPJ com API ReceitaWS

Este projeto é uma aplicação desenvolvida para realizar a validação e consulta de CNPJ, consumindo dados diretamente da API pública ReceitaWS.

A aplicação permite que o usuário insira um CNPJ válido e, a partir disso, retorna automaticamente diversas informações cadastrais da empresa, como razão social, nome fantasia, situação, endereço, entre outros dados oficiais registrados na Receita Federal.

🚀 Funcionalidades
✅ Validação de formato de CNPJ
🔎 Consulta de dados em tempo real via API ReceitaWS
📊 Exibição de informações completas da empresa:
Razão social
Nome fantasia
Situação cadastral
Data de abertura
Endereço completo
Atividade principal e secundárias
⚠️ Tratamento de erros (CNPJ inválido ou não encontrado)

🛠️ Tecnologias utilizadas
React / Next.js
Fetch / Axios para consumo de API
API ReceitaWS

📦 Como usar
Clone o repositório:
git clone https://github.com/Bvso29/validator
Instale as dependências:
npm install
Execute o projeto:
npm run dev
Insira um CNPJ válido e visualize os dados retornados.

⚠️ Observações
A API ReceitaWS possui limitações de requisições na versão gratuita.
Este projeto tem fins educacionais e demonstrativos.
